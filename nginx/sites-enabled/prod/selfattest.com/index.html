FAQ

1What is this?

I had to submit self-attested documents to GOI 3 times last week. On a normal day, this is a bearably painful exercise, but kinda unsafe during these COVID times. So, I searched online and to my surprise, there were either shady websites or sign-up walls. Hence selfattest.online was born. This is a weekend project which could not have been possible without the amazing frontend work from @Sourabh and backend was done by yours truly @mahoriR

2 Is there a limit on the number of files I can attest?

Yes. We allow only 4 files per hour.

3Is this safe?

Yes. We delete all files older than a few hours. The project source is available here
